# Screenshot Placeholders

This folder stores portfolio screenshots referenced by the main `README.md`.

Required files:
- `home.png`
- `detail-film.png`
- `detail-series.png`
- `search.png`
- `admin-dashboard.png`
- `admin-tmdb.png`
- `video-pipeline.png`

Guidelines:
- Use desktop captures at 1440px width.
- Keep dark-mode Cineclub UI visible.
- Avoid personal data in captures.
- Refresh captures after major UI changes.
